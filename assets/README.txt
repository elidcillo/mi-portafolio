# Carpeta de assets

Aquí debes colocar tus imágenes, íconos SVG y recursos gráficos para el portafolio.

Ejemplo de archivos sugeridos:
- avatar.png
- html5.svg
- css3.svg
- js.svg
- git.svg
- github.svg
- linkedin.svg
- email.svg
- proyecto1.png
- proyecto2.png
- proyecto3.png

Puedes descargar íconos gratuitos en sitios como https://simpleicons.org/ o https://www.flaticon.com/ y colocar aquí tus imágenes de proyectos y avatar.
